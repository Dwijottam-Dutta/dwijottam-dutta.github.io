About TextStorm  -  

TextStorm is one of the unique online text/notes editor with a cool UI design called MAC UI Design and has been used by many users. With TextStorm, you get a complete set of cool features like editing and modifying single text file at a time then downloading it locally, saving in cloud if you have not edited fully, transferring your text files securely to our Firebase DB and much more.
